Airplane Seat

Purpose

The purpose of this project was to practice employing multidimensional arrays. Learning to use multidimensional arrays will allow me to store data for graphic work or spreadsheat format

Description

This program simulates airplane seat reservation utilizing the class AirplaneSeat. Instances of this class are stored in a 2D array, representing an airplane seat chart. The main file provides users a menu for displaying an airplane's reserved and unreserved seats, displaying a list of passengers and their seats, searching for a specific passenger and seat, and reserving a seat.

Key Methods

• seatStatus Method

Iterates through 2D array seatName to print the name of the seat and iterates through 2D array planeSeats to the print availability of the seat

• passengerList Method

Iterates through 2D array planeSeats and prints the passenger’s information (AirplaneSeat object) if the seat is NOT available

• selectSeat Method

Prompts user to enter the passenger’s information and reserves the seat if it is available in 2D array planeSeats

• findPassenger Method

Prompts user to enter a passenger’s name and prints the passenger’s information (AirplaneSeat object) if found after iterating through 2D array planeSeats

Challenges

Displaying seat statuses in a chart was difficult because seat names were independent of the AirplaneSeat class. I resolved this by storing AirplaneSeat objects and seat names in separate but identically sized 2D arrays, accessing the same positions when printing the chart.

Improvements

Implement a 2D array of hashmaps so that seat names and AirplaneSeat objects are stored in the same data structure
